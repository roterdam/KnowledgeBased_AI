Dependencies - 

Program uses OpenCV 2.3.1 as well as numpy and scipy
sudo apt-get install libopencv-*
sudo apt-get install python-opencv
sudo apt-get install python-numpy python-scipy

Sci-kit installation instructions at 
http://scikit-image.org/download
Please run all commands with sudo privileges

You might need to install six, a python 2 and 3 compatibility library.
If needed Six installation use six-1.3.0 
Download setup from https://pypi.python.org/pypi/six/1.3.0
and run -> sudo python setup.py
